# Filoxenia Repo Notes

This is a static Filoxenia landing page.

- `index.html` is the default English page.
- `el/index.html` is the Greek page.
- `vercel.json` redirects visitors from Greece to `/el/`.
- Keep copy, layout, behavior, and asset changes in sync across both language pages unless the user explicitly asks for one locale only.
- After making changes, always prompt the user to review the rendered page in the browser before finalizing.
